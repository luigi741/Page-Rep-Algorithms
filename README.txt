The entire project is contained within the 'pageRep.cpp' file.
There is no need to include any extra files or libraries since
all of the C++ libraries are included at the top of the file.

To compile:

$: g++ pageRep.cpp -o pageRep

This will create an executable called 'pageRep' which you can run
by entering:

$: ./pageRep

From there you will have prompts that will guide you through
the program.
